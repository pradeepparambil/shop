# shop
## Product Endpoints
* POST "/api/v1/products/" - Save a new product
* PUT  "/api/v1/products/" - Update a product
* GET "/api/v1/products/" - Find all products
* GET "/api/v1/products/{id}" - Find a product
* DELETE "/api/v1/products/{id}" - Delete a product
